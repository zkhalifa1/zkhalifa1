# Hi, I'm Ziad 👋

I'm a developer who builds tools that solve real, annoying problems - mostly for myself, sometimes for others.

## What I work on

- **[FlowSwift](https://github.com/zkhalifa1/FlowSwift)** — AI-powered construction reporting. Speak your field notes, get a polished Word doc. Built with React, Firebase, and OpenAI (Whisper + GPT).
- **[Tennis Booker](https://github.com/zkhalifa1/tennis-booker)** — Chrome extension that auto-books tennis courts the moment they open. Because manually checking 13 courts at 7 AM is just not something I was ever prepared for.
- **[Outreach Automation Tool](https://github.com/zkhalifa1/Outreach_Automator)** — A local web dashboard that automates email campaigns for an architecture firm, syncing with OneDrive and sending through Outlook via Microsoft Graph API.
- **[Manulife Benefits Tracker](https://github.com/zkhalifa1/manulife_benefits_tracker)** — A clean dashboard to track your health insurance coverage, claims, and renewals — because the official app is painful.

## Stack I reach for

`React` · `TypeScript` · `Python / FastAPI` · `Firebase` · `Tailwind CSS` · `OpenAI API` · `Microsoft Graph API` · `Chrome Extensions`

## What drives me

I like the moment when a 20-step manual process becomes one click. Most of my projects start from a personal frustration - a booking flow that's too competitive, a spreadsheet that needs to be maintained manually, a voice note that should already be a report.
